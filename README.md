## 📌 Descripción

CinePlus es una aplicación web dinámica de películas desarrollada con tecnologías del frontend como HTML, CSS, Bootstrap, JavaScript (jQuery), AJAX y JSON.  

Permite visualizar una cartelera de películas con información detallada, precios dinámicos según la fecha de estreno y navegación interactiva.

---

## ⚙️ Funcionalidades

✔ Listado dinámico de películas desde JSON  
✔ Página de detalle por película  
✔ Buscador en tiempo real 🔍  
✔ Filtro por género 🎭  
✔ Sistema de precios según fecha de estreno 💰  
✔ Spinner de carga (simulación AJAX) ⏳  
✔ Visualización de tráileres 🎬  
✔ Sistema de reseñas con estrellas ⭐  
✔ Formulario de renta de películas 🧾  
✔ Formulario de contacto con validación 📩  
✔ Diseño tipo cine con temática visual 🎞️

## ▶️ Cómo ejecutar el proyecto

1. Abrir el proyecto en **Visual Studio Code**  
2. Instalar extensión **Live Server**  
3. Ejecutar `index.html` con Live Server  
4. Navegar la aplicación en el navegador

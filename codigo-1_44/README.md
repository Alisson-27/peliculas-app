## 📌 Descripción
Aplicación web dinámica de películas desarrollada con HTML, CSS, Bootstrap, jQuery, AJAX y JSON.

---

## ⚙️ Funcionalidades
- Listado dinámico de películas
- Detalle de cada película
- Buscador en tiempo real
- Filtro por género
- Sistema de precios según fecha de estreno
- Spinner de carga
- Diseño tipo cine

---

## 🚀 Tecnologías
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (jQuery)
- AJAX
- JSON

---

## ▶️ Cómo ejecutar
1. Abrir con Live Server en VS Code
2. Ingresar a `index.html`
